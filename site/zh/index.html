语音转文字，即时输入

macOS 菜单栏应用 — 按住快捷键录音，松开即转写并自动输入到当前应用。 离线优先、多引擎、AI 增强。

功能特性

macOS 上最全面的语音输入工具

🔌

离线优先

默认使用 FunASR ONNX 模型 — 无需联网，完全离线，保护隐私。

🎧

多引擎支持

FunASR（中文优化）、MLX-Whisper（99 种语言，Apple GPU 加速）、Apple Speech（系统内置）及远程 Whisper API。

✨

AI 增强

可选的 LLM 驱动文本纠错、润色、补全和翻译，支持任何 OpenAI 兼容 API。

🔗

链式模式

多步骤增强管道 — 例如先纠错润色再翻译 — 通过 Markdown 文件灵活定义。

📚

个人词库

基于向量检索的个人词库，从纠正历史中学习专有名词和领域术语，越用越准确。

⚡

轻量级

仅在菜单栏运行，不占 Dock 位置。资源占用极小，快捷键即时响应。

使用方式

三步操作，无需切换窗口

1

按住录音

按住 fn 键（或自定义快捷键）开始说话。浮动指示器显示录音状态和音量条。

2

松开转写

松开按键 — VoiceText 使用选定的语音引擎转写语音，可选 AI 增强处理。

3

自动输入

转写结果自动输入到当前活动应用，或在预览面板中显示供编辑和确认。

预览面板 — 查看、编辑、切换模式、对比模型，确认后再输出

语音识别引擎

根据语言和硬件选择最佳引擎

引擎 语言 速度 准确度 下载大小
FunASR 默认 中文 快 高 ~500 MB
MLX-Whisper 99 种语言 中等 高 75 MB – 1.6 GB
Apple Speech 多语言 快 良好 无需下载（系统内置）
Whisper API 多语言 取决于网络 高 无需下载（云端）

安装方式

三种方式，任选其一

推荐

下载安装

最简单的方式。无需 Python、无需构建工具 — 下载、拖入、启动，即刻使用。

⬇ 下载最新版本
1 从 Releases 下载 VoiceText.app
2 拖入 /Applications
3 双击启动
4 授予麦克风和辅助功能权限

首次启动：macOS 会阻止未签名应用。前往系统设置 → 隐私与安全性，点击仍要打开即可。

🔨

从源码构建

需要 Python 3.13+ 和 uv

git clone https://github.com/Airead/VoiceText
cd VoiceText
uv sync
./scripts/build.sh
💻

从源码运行

用于开发和调试

git clone https://github.com/Airead/VoiceText
cd VoiceText
uv sync
uv run python -m voicetext
