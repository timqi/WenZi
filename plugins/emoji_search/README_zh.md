# Emoji 搜索

通过 WenZi 启动器搜索并粘贴 Emoji。

## 使用方法

在启动器中输入 `e ` 加关键字：

- `e kai xin` → 😀
- `e mao` → 🐱
- `e huo che` → 🚄

按 **回车** 将选中的 Emoji 粘贴到当前应用。  
按 **Alt+回车** 将其复制到剪贴板。

### 按分组搜索

使用 `@分组名` 将搜索范围限制在某个分类或子分类下。分组名支持模糊匹配，不需要完整输入。

- `e @动物` → "动物与自然" 分类下的全部 Emoji
- `e @face eye` → 在 "face-*" 子分类中搜索 "eye"
- `e mao @dongwu` → 在匹配 "dongwu" 的分组中搜索 "mao"

如果分组过滤没有命中任何分组，搜索会自动回退到全局范围。

### 分组过滤的解析规则

插件会从 `@` 后面由前往后逐个词尝试匹配已知的分组/子分组名称，直到命中为止。因此支持带空格的多词分组名，无需加引号。

## 数据来源

Emoji 数据改编自 [angelofan/emoji-json](https://github.com/angelofan/emoji-json)，提供简体中文翻译并与 Unicode 联盟发布的数据保持同步。
