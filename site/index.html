Features

Everything you need for voice input on macOS

🔌

Offline-First

Uses FunASR ONNX models by default — no cloud dependency, full privacy. Works without an internet connection.

🎧

Multi-Engine

FunASR (Chinese), MLX-Whisper (99 languages, Apple GPU), Apple Speech (built-in), and remote Whisper API.

✨

AI Enhancement

Optional LLM-powered proofreading, formatting, completion, and translation via any OpenAI-compatible API.

🔗

Chain Modes

Multi-step enhancement pipelines — e.g., proofread then translate — defined via simple Markdown files.

📚

Personal Vocabulary

Embedding-based vocabulary retrieval learns your proper nouns and domain terms from correction history.

⚡

Lightweight

Runs as a menubar-only app — hidden from Dock. Minimal resource usage with instant hotkey response.

How It Works

Three simple steps — no window switching needed

1

Hold to Record

Hold the fn key (or your custom hotkey) and speak. A floating indicator shows recording status with audio level bars.

2

Release to Transcribe

Release the key — VoiceText transcribes your speech using the selected ASR engine, optionally enhanced by AI.

3

Text is Typed

The result is automatically typed into the active application, or shown in a preview panel for review and editing.

The Preview panel — review, edit, switch modes, and compare models before confirming

Installation

Three ways to get started

Recommended

Download Release

The easiest way to get started. No Python, no build tools — just download, drag, and go.

⬇ Download Latest Release
1 Download VoiceText.app from Releases
2 Drag to /Applications
3 Double-click to launch
4 Grant Microphone & Accessibility permissions

First launch: macOS blocks unsigned apps. Go to System Settings → Privacy & Security and click Open Anyway.

🔨

Build from Source

Requires Python 3.13+ and uv

git clone https://github.com/Airead/VoiceText
cd VoiceText
uv sync
./scripts/build.sh
💻

Run from Source

For development and debugging

git clone https://github.com/Airead/VoiceText
cd VoiceText
uv sync
uv run python -m voicetext
