# v0.2.0

## New Features

- **LLM Provider Presets**: The LLM settings panel now includes preset configurations for popular providers. When adding a new provider, you can select from a dropdown of pre-configured services:
  - OpenAI (gpt-4o, gpt-4o-mini, etc.)
  - Qwen (通义千问) - qwen-max, qwen-plus, qwen-turbo, etc.
  - Doubao (豆包) - doubao-1-5-pro-32k, doubao-1-5-lite-32k, etc.
  - DeepSeek - deepseek-chat, deepseek-reasoner
  - MiniMax - MiniMax-Text-01, abab6.5s-chat, etc.
  - Zhipu (智谱AI) - glm-4-plus, glm-4-9b, glm-4-flash, etc.
  - Moonshot (月之暗面) - moonshot-v1-8k, moonshot-v1-32k, etc.
  - SiliconFlow - DeepSeek-V3, DeepSeek-R1, Qwen2.5-72B, etc.
  - Ollama (Local) - qwen2.5:7b, deepseek-r1:7b, llama3.2:3b, etc.
  
  Selecting a preset auto-fills the base URL and shows a dropdown of available models for easy selection.

## Improvements

- **Easier LLM Provider Setup**: When adding an LLM provider, selecting a preset automatically fills in the base URL and provides a model dropdown. This makes it much easier for users familiar with Chinese AI services to get started.
