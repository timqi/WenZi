Speech to Text, Instantly

A macOS menubar app — hold a hotkey to record, release to transcribe and type. Offline-first, multi-engine, AI-enhanced.

Features

Everything you need for voice input on macOS

🔌

Zero Setup

Apple On-Device speech recognition works out of the box — no downloads, no API keys. Switch to FunASR or MLX-Whisper for higher accuracy when needed.

🎧

Multi-Engine

Five ASR backends: Apple Speech (built-in), FunASR (Chinese), MLX-Whisper (99 languages, Apple GPU), Sherpa-ONNX, and remote Whisper API.

✨

AI Enhancement

Optional LLM-powered proofreading, formatting, completion, and translation via any OpenAI-compatible API.

🔗

Chain Modes

Multi-step enhancement pipelines — e.g., proofread then translate — defined via simple Markdown files.

📚

Personal Vocabulary

Embedding-based vocabulary retrieval learns your proper nouns and domain terms from correction history.

🎤

Real-time Streaming

Live transcription overlay shows partial results as you speak — see your words appear in real time during recording.

📅

History & Statistics

Browse past transcriptions with tag filters, time ranges, and deletion. Interactive stacked bar charts visualize your usage patterns.

💻

Scripting System

Python-based plugin system with leader keys and hotkeys for macOS automation — alerts, clipboard, shell commands, and more.

🌑

Dark Mode

Full dark mode support across all panels and windows. UI adapts automatically to your macOS appearance setting.

⚡

Lightweight

Runs as a menubar-only app — hidden from Dock. Minimal resource usage with instant hotkey response.

How It Works

Three simple steps — no window switching needed

1

Hold to Record

Hold the fn key (or your custom hotkey) and speak. A floating indicator shows recording status with audio level bars, and a live transcription overlay displays partial results in real time. Press Space to restart, ESC or Fn+Cmd to cancel.

2

Release to Transcribe

Release the key — WenZi transcribes your speech using the selected ASR engine, optionally enhanced by AI.

3

Text is Typed

The result is automatically typed into the active application, or shown in a web-based preview panel for review, editing, and model comparison.

The Preview panel — review, edit, switch modes, and compare models before confirming

Installation

Three ways to get started

Recommended

Download Release

The easiest way to get started. No Python, no build tools — just download, drag, and go.

⬇ Download Latest Release
1 Download WenZi.app from Releases
2 Drag to /Applications
3 Double-click to launch
4 Grant Microphone & Accessibility permissions

First launch: macOS blocks unsigned apps. Go to System Settings → Privacy & Security and click Open Anyway.

🔨

Build from Source

Requires Python 3.13+ and uv

git clone https://github.com/Airead/WenZi
cd WenZi
uv sync
./scripts/build.sh
💻

Run from Source

For development and debugging

git clone https://github.com/Airead/WenZi
cd WenZi
uv sync
uv run python -m wenzi
